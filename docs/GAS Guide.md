# 📊 Google Sheets & Google Apps Script 연동 가이드

이 문서는 학생들이 모바일, 태블릿, 크롬북(디벗) 환경에서 제출한 시뮬레이션 최종 보고서 데이터를 구글 스프레드시트에 실시간으로 누적 저장하고, 교사용 대시보드로 다시 조회하기 위한 백엔드 설정 매뉴얼입니다.

---

## 🛠️ Step 1. 구글 스프레드시트 및 스크립트 생성

1. **구글 스프레드시트**([Google Sheets](https://sheets.google.com))에 접속하여 새로운 스프레드시트를 생성합니다.
2. 스프레드시트 주소창에서 `d/`와 `/edit` 사이에 위치한 **긴 문자열(스프레드시트 ID)**을 드래그해서 따로 복사해 둡니다.
   * 예: `https://docs.google.com/spreadsheets/d/1A_B_C_D_E_F_12345/edit` 주소라면 `1A_B_C_D_E_F_12345`가 ID입니다.
3. 상단 메뉴에서 **[확장 프로그램] ➡️ [Apps Script]**를 클릭하여 코드 편집기를 엽니다.
4. 기본 코드를 모두 지우고, [Code.gs](file:///c:/Z-바이브/MayorSimulator/docs/Code.gs)의 소스코드를 복사하여 그대로 붙여넣습니다.
5. 코드 **11번째 라인**의 `const SPREADSHEET_ID = "YOUR_SPREADSHEET_ID_HERE";` 부분을 찾아서, 아까 복사해 둔 스프레드시트 ID를 붙여넣고 저장합니다.
   * 예: `const SPREADSHEET_ID = "1A_B_C_D_E_F_12345";`
6. 상단의 디스켓 모양 아이콘 **[프로젝트 저장]**(또는 `Ctrl + S`)을 누릅니다.

---

## 🚀 Step 2. 웹 앱 배포 설정 (중요)

구글 보안 권한상 배포 시 옵션 선택을 반드시 아래 가이드와 일치시켜야 외부 학생들이 접근할 수 있습니다.

1. Apps Script 편집기 우측 상단의 파란색 **[배포] ➡️ [새 배포]** 버튼을 클릭합니다.
2. 새 배포 팝업창에서 왼쪽 톱니바퀴 아이콘을 누르고 유형이 **[웹 앱]**으로 선택되어 있는지 확인합니다.
3. 아래의 설정을 정확하게 맞춰줍니다:
   * **설명:** `내가 만약 시장이라면 데이터 저장소` (자유롭게 입력 가능)
   * **웹 앱을 실행할 사용자:** **[나(본인의 구글 계정)]**으로 설정합니다.
   * **액세스할 수 있는 사용자:** **[모든 사용자(Anyone)]**로 설정합니다. (로그인하지 않은 외부 학생 기기에서도 데이터를 전송해야 하므로 반드시 '모든 사용자'여야 합니다.)
4. **[배포]** 버튼을 클릭합니다.
5. 처음 배포하는 경우 구글 계정 접근 승인을 요구하는 창이 뜹니다. **[액세스 승인]**을 누르고, 본인 계정을 선택한 뒤 **[Advanced(고급)] ➡️ [Go to Untitled project (unsafe)]** 링크를 클릭하여 최종 권한 허용을 완료해 줍니다.

---

## 🔗 Step 3. 배포 URL 복사 및 연동

1. 배포가 완료되면 화면에 **웹 앱 URL**이 생성됩니다. (`https://script.google.com/macros/s/.../exec` 형식)
2. 해당 URL을 복사(Copy)합니다.
3. 우리 프로젝트의 [app.js](file:///c:/Z-바이브/MayorSimulator/app.js) 파일을 열어 **4번째 라인**을 찾습니다:
   ```javascript
   const GAS_WEB_APP_URL = "YOUR_GAS_WEB_APP_URL_HERE";
   ```
4. `"YOUR_GAS_WEB_APP_URL_HERE"` 영역을 지우고 복사한 **웹 앱 URL**을 붙여넣고 저장합니다.
   ```javascript
   const GAS_WEB_APP_URL = "https://script.google.com/macros/s/AKfycb.../exec";
   ```

---

## 🔑 Step 4. 교사용 관리자 대시보드 사용 방법

1. 메인 로그인 화면 우측 하단의 **[🔑 교사용 관리자 대시보드 바로가기]** 링크를 클릭합니다.
2. 관리자 인증 화면으로 전환되면 아이디 **`admin`**과 비밀번호 **`admin1204`**를 입력하고 로그인 버튼을 누릅니다.
3. 인증에 성공하면 학생들의 누적 결과 리스트 대시보드로 진입합니다.
4. **대시보드 기능:**
   * **새로고침:** 구글 시트에서 최신 학생 제출 데이터를 동적으로 실시간 동기화하여 가져옵니다.
   * **조회:** 해당 학생의 행 우측 **[조회]** 버튼을 클릭하면, 그 학생이 제출했던 **최종 지표(인구/행복/예산), 선 그래프 3개, 임기 중 결재했던 상세 역사 타임라인, 에세이 소감 답변**이 고스란히 화면에 복원되어 표시됩니다.
   * **조회 후 동작:** 보고서 복원 상태에서 **[보고서 인쇄하기]** 버튼을 누르면 해당 학생의 A4 가로 최적화 포트폴리오를 즉시 종이로 인쇄하거나 PDF로 저장할 수 있습니다.
   * **대시보드로 돌아가기:** 조회 뷰 하단의 파란색 대시보드로 돌아가기 버튼을 누르면 원본 학생 목록으로 복귀합니다.
