# 🗺️ [구현 계획] '내가 만약 시장이라면' 화면 구성 및 UI 시각화

본 계획은 초등학교 고학년용 도시 경영 시뮬레이션 게임 앱 **'내가 만약 시장이라면'**의 화면 구성을 위한 마크업, CSS 스타일링, 그리고 인터랙티브 스크립트의 뼈대를 설계하기 위한 것입니다.

---

## 📌 1. 개발 범위 및 접근 방식

* **프레임워크 배제 (Vanilla Stack):** 초등 디벗(크롬북 등)의 사양과 구동 안정성을 고려하여, 복잡한 빌드 과정이 없는 **순수 HTML5, CSS3, JavaScript** 구조로 개발합니다.
* **싱글 페이지 애플리케이션 (SPA) 구조:** 화면 간의 지연 없는 부드러운 전환 효과(페이드인, 슬라이드 등)를 위해 한 페이지 내에서 CSS 클래스 조작을 통해 화면(Screen)을 전환합니다.
* **더미 데이터 및 흐름 제어:** 기획서에 명시된 5개년 진행 루틴을 실제 체감할 수 있도록 모의 정책 목록과 가상 연산 로직을 포함한 프론트엔드 목업을 구축합니다.

## 🎨 2. 디자인 시스템 및 시각화 계획

* **글꼴 연동 (Google Fonts):**
  * `Nanum Square Round` 대체용으로 둥글고 친근한 분위기의 **Google Fonts `Gaegu`** 또는 **`Nunito`** (혹은 웹 폰트 링크 수급)를 적용하거나 로컬 서체 선언을 정교화합니다. (Pretendard는 CDN 연동)
* **CSS 변수 선언 (`style.css`):**
  ```css
  :root {
    --primary-color: #4A90E2;    /* Soft Blue (신뢰) */
    --positive-color: #2ECC71;   /* Mint Green (성장) */
    --negative-color: #E74C3C;   /* Soft Coral (경고) */
    --bg-color: #F8F9FA;         /* 기본 눈피로 방지 연회색 */
    --card-bg: #FFFFFF;
    --text-main: #2C3E50;
    --text-sub: #7F8C8D;
    --border-radius: 16px;
    --shadow-soft: 0 10px 30px rgba(0, 0, 0, 0.05);
  }
  ```
* **인터랙션 및 애니메이션:**
  * 정책 실행 시 **비서실 서류 도장 날인 애니메이션** (Scale Up & Rotation)
  * 신문 기사 결과 보고서 팝업의 **슬라이드업 페이드인** 효과
  * 마우스 호버 시 카드 UI의 **미세한 공중 부양(Lift Up) 및 그림자 확장** 효과

## 🖥️ 4. 화면별 구현 상세 계획

### **Step 1. 로그인 화면**
* 시장 이름 입력창 및 도시 이름 입력창 배치
* **`🎲 도시명 추천`** 버튼 클릭 시, 동적으로 도시 이름 조합(예: 푸른샘, 미리내 등)을 텍스트 상자에 입력

### **Step 2. 도시 규모 선택 화면**
* 대도시(복잡함), 중간 도시(적당함), 소도시(한적함)를 나타내는 3개의 반응형 카드 배치
* 마우스 호버 시 줌인 효과 및 테두리 강조선 적용
* 카드 하단 클릭 시 해당 도시 데이터가 `app.js` 상태에 바인딩되고 메인 게임으로 이동

### **Step 3. 메인 게임 화면 (3단 레이아웃)**
* **좌측 (20%):** 인구, 예산, 행복도를 시각적인 게이지 또는 카드 형태로 구현. 인구 밀도에 따라 빨강(대도시)/노랑(중간도시)/초록(소도시) 신호등 뱃지 부착.
* **중앙 (50%):** 5개의 공공 정책 리스트를 가로/세로 카드 리스트로 정렬. 클릭 시 활성화 상태 표시.
* **우측 (30%):** 선택된 정책에 대한 비서의 결재 브리핑 문서 디자인 (종이 서류 느낌의 연출). 우측 하단에 `[정책 실행 결재하기 🖋️]` 버튼 강조.

### **Step 4. 정책 시행 결과 (신문 팝업)**
* 결재 버튼 클릭 시 **도장 이펙트** 연출
* 이후 모달 형태의 신문 지면이 나타나며 가상 헤드라인과 상승/하강(`📈`, `📉`) 애니메이션 효과가 반영된 지표 변동 표시.

### **Step 5. 시정 결과 엔딩 보고서**
* 5년 임기 완료 후 최종 획득한 등급 타이틀(예: 황금 밸런스 시장)과 별점 노출.
* **Chart.js**를 활용하여 5년간의 `인구`, `예산`, `행복도`를 나타내는 선 그래프 3개 렌더링.
* 오프라인 포트폴리오를 위한 **`[인쇄하기 🖨️]`** 버튼 및 **`[선생님께 제출하기 📤]`** 버튼 구현 (제출 성공 모의 팝업 포함).

1. **FHD 해상도 최적화 확인:** 브라우저 개발자 도구를 이용하여 1920x1080 및 기타 표준 데스크톱 해상도에서의 3단 레이아웃 깨짐 여부 검사.
2. **5개년 시정 시뮬레이션 완주 테스트:** 로그인 ➡️ 도시 선택 ➡️ 5년 경영(정책 선택 및 결과 확인 루틴) ➡️ 엔딩 리포트 생성 및 그래프 렌더링이 중단 없이 부드럽게 흐르는지 확인.
3. **인쇄 테스트:** 브라우저 인쇄 단축키(Ctrl+P) 혹은 인쇄 버튼을 눌러 결과 화면에서 불필요한 컨트롤이 숨겨지고 A4 용지에 알맞게 정렬되는지 인쇄 미리보기 검증.
