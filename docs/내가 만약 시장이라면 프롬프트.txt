<내가 만약 시장이라면(If I Were Mayor) 프롬프트>


[프롬프트 #01]
  초등학생 고학년 사회 시간에 활용할 '내가 만약 시장이라면' 시뮬레이션 앱을 만들려고 합니다. 대도시, 중간 도시, 소도시 중 하나를 선택해 시장으로 취임하고, 5년간 시를 운영하는 시뮬레이션 앱입니다.
  - 도시 규모에 따라 예산(1000억, 500억, 100억)과 인구(50만, 30만, 10만 명) 등이 다르고
  - 매년 2~3개 정도의 정책을 시장이 선택합니다.(정책에 따라 사용하는 예산이 다름)
  - 정책은 기업 유치를 통한 일자리 증가, 교육시설 확보, 생활 편의 시설 확보, 환경 문제 해결, 교통 시설 개선, 안전 문제 등과 관련된 것을 제시하고 학생들이 각자 선택
  - 학생들의 선택에 따라 예산이 사용되고, 인구의 증감이 발생하고, 시민들의 행복도에 변화가 생김
  - 5년간 시장의 역할을 마치면 도시의 상태를 보여 주고, 시장 평가 보고서를 작성해서 보여줌

  이 정도를 구상해 보았는데 아이디어를 다듬어서 더 좋은 제안을 해 주세요.

----------

[프롬프트 #02]
  초등학교 사회책에 나오는 국가의 균형 발전을 위한 정책을 몇 가지 소개하니 기획서를 작성할 때 참고해 주세요.
  1. 공공 기관 이전 : 공공 기관을 수도권에서 전국 곳곳으로 옮겨 인구 이동과 지역 발전을 유도
  2. 로컬 브랜딩 사업 지원 : 지역의 자원과 문화 등을 활용하여 지역만의 매력을 만드는 사업을 지원해 사람들을 끌어 모읍니다.
  3. 고향 사랑 기부제 시행 : 고향이나 관심 지역에 기부하면 답례품과 세금 혜택을 주는 제도를 시행
  4. 청년 마을 만들기 : 청년 마을을 만들어 정착을 지원함으로써 지역 청년의 유출을 줄입니다.
  5. 학교, 학원 등 교육 시설 확보
  6. 병원, 도서관, 백화점, 공연장 등 편의/문화 시설 확보
  7. 교통, 환경 문제 개선

----------

[프롬프트 #03]
  시뮬레이션의 진행은 다음과 같이 하고 싶은데 괜찮을까요?

  1. 초기 화면 : 간단한 로그인 
    - 시장님의 이름과 도시 이름을 입력(도시 이름을 입력하지 않으면 몇 가지 가상의 도시 이름 추천)

  2. 도시 규모 선택 화면
    - 대도시, 중간 도시, 소도시 중 한 가지 선택
    - 각 도시의 특징을 간단히 소개
    - 도시의 시작 인구, 예산, 현재 시민 행복도 등을 표시
    - 각 선택 하단에는 예를 들어 '중간 도시의 시장으로 취임하기' 버튼

  3. 메인 게임 화면
    - 3단 구조
    - 왼쪽 패널 : 도시 이름, 시장 이름, 현재 인구수, 현재 예산, 현재 인구 밀도, 시민 행복도 등을 표시
    - 가운데 패널 : 임기 ○년차 / 202○년 추친 정책 / "○○○ 시장님! 아래 5가지 정책 중 한 가지를 선택해 주세요. 오른쪽 패널에서 각 정책을 시행했을 때 나타나는 효과와 예산을 확인한 후 최종 결재를 해 주세요." / 아래에 5가지 정책 제시
    - 오른쪽 패널 : 가운데 패널에서 시장이 정책을 선택하면 간단한 설명, 인구 변동, 행복도 증가 등의 효과, 필요한 예산 등을 표시 / 밑에는 '정책 결재하기' 버튼

  4. 3번 화면에서 '정책 결재하기' 버튼을 클릭하면 2~3초 후 '임기 ○년차 ○○ 정책 시행 결과 보고서'를 한 화면에 간단하게 표시. 내용으로는 정책 내용, 인구 증감, 시민들의 행복도 증감, 예산 증감, 가상의 뉴스 기사 등을 포함 

  5. 5년의 임기 동안 매년 깜짝 이벤트를 포함하여 2~3건의 정책을 선택

  6. 5년 임기가 끝나면 '○○○ 시장님의 시정 결과 보고서' 표시
    - 보고서에는 최종 등급, 간단한 칭찬(예. 환경 문제 전문가 등), 분야별(인구밀도, 시민행복도, 예산집행 등) 평가 점수를 표시
    - 그 밑에는 퇴임 시 최종 인구(시작 인구와 비교하여 증감 표시), 시민들의 행복도, 예산 현황 등을 표시
    - 그 밑에는 5년간 인구 밀도 변화 그래프, 시민 행복도 변화 그래프, 예산 변화 그래프 표시
    - 그 밑에는 시장님의 5년 임기 동안 의사 결정 이력 타임라인 표시
    - 끝으로 시뮬레이션 학습을 마무리 하면서 정리하는 질문 제시(3개 정도, 아래 예시)
      (1) 국가의 균형 발전을 위해 가장 중요한 정책이 무엇이라고 생각하나요?
      (2) 왜 그렇게 생각하나요?
      (3) 5년 간 시장을 맡아 시를 운영하고 느낀 점을 적어봅시다.

  모든 내용을 작성할 때 초등학생 고학년임을 고려하여 알맞은 어휘를 사용하고 전문적이거나 어려운 내용은 쉽게 풀이해 줘야 합니다.

[프롬프트 #04]
  이 앱은 시뮬레이션이라 화려한 그래픽은 필요 없고 텍스트 위주의 화면이 대부분일 것 같습니다.
  - 학생들이 사용하는 디벗(크롬북)의 해상도(FHD, 1920x1080)를 고려하여 화면 구성
  - 한 화면에 글씨가 너무 많이 포함되지 않도록 조절(설명이 지나치게 길지 않도록 조절 필요)
  - 글꼴은 Nanum Square Round와 Pretendard를 사용
  - 텍스트 위주의 앱이지만 너무 딱딱하지 않게 곳곳에 귀여운 아이콘 표시

  이러한 조건을 반영하여 앱의 디자인 톤과 매너를 제안해 주세요.
